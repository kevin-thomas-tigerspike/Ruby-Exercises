Exercise 4: Pitch

Your Task
In music, each note is named by its pitch class (e.g. C, E♭, F♯), and each pitch class can alternatively be expressed as an integer from 0 to 11. Your task is to write a method pitch_class that, when give a letter-based pitch class, returns the corresponding integer.

Only seven letters are used to name the notes: "A" through "G". These letter names are cyclical, just like the days of the week. The notes corresponding to those letters are called the "natural notes." Here are the numbers corresponding to each of them:

  • C : 0
  • D : 2
  • E : 4
  • F : 5
  • G : 7
  • A : 9
  • B : 11

So pitch_class('D') should return 2, and pitch_class('B') should return 11.


    > pitch_class('D')
    => 2

    > pitch_class('B')
    => 11

The share sign ("♯") is essentially an increment operator, so "C♯" (pronounced "C sharp") refers to one note higher than C, which has a value of 1, whereas F♯ has a value of 6. Instead of using the sharp sign, use a number sign ("#") instead.

The flat sign ("♭") is the opposite of a sharp, meaning one note lower. F♭ has a value of 4, and C♭ has a value of 11 (the twelve-note system is cyclical). Instead of using the flat sign, use a lowercase "b" instead.
